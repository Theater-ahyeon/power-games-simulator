# 权力游戏：政治阴谋的艺术

## 项目概述

权力游戏（Power Game）是一款以权力斗争为核心机制的深度策略游戏。玩家将置身于一个架空的现代政治世界中，管理稀缺资源、构建势力联盟、收集情报信息、进行政治博弈，并在关键时刻做出影响国家命运的重大决策。

### 游戏特色

- **沉浸式政治模拟**：体验真实的权力运作机制，从资源分配到联盟博弈
- **复杂人物系统**：22位性格各异的NPC，每个角色都有独特的政治立场、隐藏动机和个人目标
- **多线叙事架构**：8个核心剧情节点，每个节点提供3至5个分支选项
- **战略决策深度**：多层博弈树系统，决策效果可能在数回合后才显现
- **八种结局设计**：从独裁统治到民主改革，从幕后操控到流亡海外

## 核心系统

### 一、资源管理系统

资源是权力的基础。本游戏包含8种核心资源类型：

- **政治资本**：衡量政治影响力的核心指标
- **势力影响力**：在各派系中的话语权
- **情报信息**：情报网络的核心资源
- **财富资源**：经济实力的体现
- **军事力量**：武装力量的控制权
- **声望声誉**：公众形象和权威性
- **合法性**：执政的正当性基础
- **民众支持**：基层民意基础

资源之间可以相互转换，但存在效率曲线；同时，某些资源变化会产生延迟效果，模拟现实政治的滞后性。

### 二、派系与忠诚系统

游戏中有5大政治派系：

| 派系 | 标识 | 特点 | 核心属性 |
|------|------|------|----------|
| 保守联盟 | 传统派 | 传统权力持有者，军队背景深厚 | 高财富、高军事 |
| 自由联盟 | 改革派 | 改革倡导者，公众支持广泛 | 高声誉、高诚信 |
| 中间集团 | 平衡派 | 务实的调停者，资源均衡 | 全属性平衡 |
| 激进阵线 | 变革派 | 革命力量，野心勃勃 | 高谋略、高野心 |
| 机会主义圈 | 投机派 | 利益导向，情报能力突出 | 高谋略、高智慧 |

每个派系成员都有忠诚度、信任度和恐惧值三个维度。系统会计算角色叛变的概率。

### 三、情报收集系统

情报是权力游戏的命脉：

- **间谍部署**：派遣特工渗透对手派系
- **情报网络**：建立和管理信息收集网络
- **情报分析**：评估情报可靠性和反情报风险
- **反情报行动**：安全排查和反间谍行动

### 四、联盟与背叛系统

8种条约类型构建复杂的国际和国内关系网：

- 军事同盟
- 经济联盟
- 政治联合
- 情报共享
- 技术合作
- 贸易协定
- 互不侵犯
- 全面战略伙伴

信任水平会根据行动和事件动态变化。背叛行为将产生连锁反应，影响与其他所有势力的关系。

### 五、角色系统

22位独特NPC，每位角色具备：

- **五种政治立场**：保守派、自由派、中立派、激进派、机会主义者
- **八项核心属性**：谋略、魅力、智慧、财富、军事、声望、诚信、野心
- **隐藏动机**：个人目标、秘密关系、背叛倾向
- **AI决策引擎**：基于行为树和效用计算的智能决策系统

### 六、剧情系统

8个关键剧情节点，每个节点包含3至5个分支选择：

```
权力真空 → 派系角力 → 情报战争 → 经济危机 → 外国干预
                                    ↓
                              宪法危机 → 关键抉择
                                    ↓
          ┌─────────────┬───────────┼───────────┬─────────────┐
          ↓             ↓           ↓           ↓             ↓
      紧急状态      宪法改革     达成妥协     军事政变     流亡海外
          ↓             ↓           ↓           ↓             ↓
        独裁        民主改革      权力平衡    系统崩溃    外国扶植
```

### 七、策略决策系统

- **决策树**：多层影响链，包含3层以上的因果关系
- **延迟后果**：某些效果在数回合后才显现
- **决策分析**：短期收益、长期影响和风险评估
- **概率结果**：基于计算概率的多样化结果

### 八、多结局系统

8种截然不同的结局：

1. **独裁统治**：掌握所有权力，建立极权政体
2. **民主改革**：推动制度改革，建立民主体制
3. **权力平衡**：多方制衡，维持现状
4. **流亡海外**：失去权力，被迫流亡
5. **幕后操控**：在暗处操控傀儡政权
6. **系统崩溃**：政治秩序彻底瓦解
7. **外国干预**：引入外部势力介入
8. **个人野心**：牺牲一切实现个人野心

| 历史事件 | 游戏应用 |

| 水门事件 | 情报战节点中的监控与掩盖机制 |

## 安装指南

### 系统要求

- **操作系统**：Windows 10/11、Linux、macOS
- **编译器**：C++17兼容（MSVC 2019+、GCC 9+、Clang 10+）
- **构建工具**：CMake 3.20+
- **磁盘空间**：至少500MB可用空间
- **内存**：推荐8GB以上

### 构建步骤

#### Windows系统（使用MSVC）

```bash
# 克隆仓库
git clone https://github.com/Theater-ahyeon/power-games-simulator.git
cd power-games-simulator

# 使用CMake配置并构建
cmake -B build -G "Visual Studio 17 2022"
cmake --build build --config Release

# 或使用项目提供的构建脚本
.\build.bat
```

#### Linux/macOS系统

```bash
# 克隆仓库
git clone https://github.com/Theater-ahyeon/power-games-simulator.git
cd power-games-simulator

# 使用CMake配置并构建
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build

# 或使用项目提供的构建脚本
chmod +x build.sh
./build.sh
```

### 依赖说明

项目使用CMake的FetchContent功能自动下载nlohmann_json库，无需手动安装第三方依赖。

## 使用方法

### 运行游戏

构建完成后，可执行文件位于：

```bash
# Windows
.\build\bin\PowerGame.exe

# Linux/macOS
./build/bin/PowerGame
```

### 运行测试

```bash
# Windows
.\build\bin\PowerGameTests.exe

# Linux/macOS
./build/bin/PowerGameTests
```

### Python工具（可选）

```bash
# 安装依赖
pip install numpy

# 运行数据分析工具
python Tools/DataAnalyzer/data_analyzer.py

# 运行AI训练工具
python Tools/AITrainer/ai_trainer.py
```

### 游戏操作指南

1. **主菜单**：新建游戏、加载存档、查看设置、退出游戏
2. **回合流程**：每个回合包含行动阶段、结算阶段和回合结束阶段
3. **资源管理**：通过资源面板查看和分配资源
4. **派系交互**：与其他派系进行谈判、结盟或对抗
5. **情报收集**：部署间谍网络，收集对手信息
6. **剧情决策**：在关键节点做出选择，影响游戏走向

## 美术资源规范

本项目的美术资源严格遵循"主权情报"设计系统（The Sovereign Intelligence Design System）：

### 设计原则

- **冷峻美学**：模拟高级别战争指挥室的冷硬环境
- **建筑不对称**：采用权重侧边栏和模块化数据集群
- **高对比度排版**：衬线字体与几何无衬线字体的对比运用
- **零圆角设计**：所有元素采用0px圆角，强化战争室美学

### 色彩体系

- **主背景**：#111316（深空灰）
- **次级背景**：#0c0e11（近黑）
- **强调色一**：#00daf3（青色，信任/技术）
- **强调色二**：#e9c349（金色，威望/重要）
- **警告色**：#ffb4ac（绯红，威胁/冲突）

### 排版规范

- **标题字体**：Newsreader（衬线体，权威感）
- **正文字体**：Inter（无衬线，高可读性）
- **数据字体**：Space Grotesk（等宽，数据展示）

### 文件命名规范

所有美术资源文件采用以下命名规范：

```
<类别>_<具体内容>_<变体>.txt
```

示例：

- `title_screen.txt`（标题画面）
- `main_dashboard.txt`（主仪表盘）
- `resource_panel.txt`（资源面板）
- `intelligence_network.txt`（情报网络）
