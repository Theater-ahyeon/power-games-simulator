Game Overview
Power Game is a political strategy simulator set in a fictional modern society. Players navigate complex political landscapes, managing resources, building factions, gathering intelligence, forming alliances, and making critical decisions that shape the nation's destiny. Every choice carries consequences that ripple through the political ecosystem.

Core Systems
1. Resource Management System (Political Capital, Influence, Information, Wealth, Military, Reputation, Legitimacy, Public Support)
2. Faction & Loyalty System (Conservative Alliance, Liberal Coalition, Central Bloc, Radical Front, Opportunist Circle)
3. Intelligence Collection System (Spy Deployment, Intel Networks, Counter-Intelligence)
4. Alliance & Betrayal System (8 Treaty Types, Dynamic Trust, Betrayal Mechanics)
5. Character System (22+ Unique NPCs, AI Decision Engine)
6. Story System (8 Key Story Nodes, Dynamic Generation)
7. Strategy & Decision System (Decision Trees, Delayed Consequences)
8. Multi-Ending System (8 Distinct Endings)

Game Flow
Start Game → Power Vacuum Event → [Choose Path] → Intelligence War → Economic Crisis → Foreign Interference → Constitutional Crisis → [Critical Choice] → Final Gambit → Ending Evaluation.
