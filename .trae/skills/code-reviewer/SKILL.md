# Code Reviewer

This skill performs comprehensive code reviews to improve code quality.

## Review Areas

- **Code Quality**: Readability, maintainability, style consistency
- **Performance**: Efficiency, memory usage, algorithmic complexity
- **Security**: Vulnerability detection, secure coding practices
- **Testing**: Test coverage, test quality
- **Documentation**: Code comments, API documentation
- **Design Patterns**: Proper architecture, SOLID principles

## Review Process

1. Code analysis and pattern identification
2. Potential bug detection
3. Performance bottleneck identification
4. Security vulnerability scanning
5. Best practice recommendations
6. Refactoring suggestions

## Usage

Invoke this skill when:
- Before merging pull requests
- After completing major features
- When experiencing bugs
- For performance optimization
- Security audits
- Code quality improvements

## Output Format

Provide detailed feedback with:
- Issue severity (Critical, High, Medium, Low)
- Line-specific comments
- Suggested fixes
- Code examples
- References to best practices
